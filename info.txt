METODOS COMUNS

- GET - Buscar uma informacao
- PUT - Alterar uma informacao
- POST - Inserir uma informacao
- DELETE - remover uma informacao
- PATCH - Alterar PONTUAL de uma informacao

Os Tipos de parametros
- Body - { }
- Query Params - http://localhost:8085/users?nome=Daniele&idade=34 OPCIONAIS
- Parametros de rotas - http://localhost:8085/users/{nome}/{idade}
    
    http://localhost:8085/users/ = Assim vai chamar 

Explica de uma maneira curta pratica

ORM -> Object Relational Mapper 

Repositorio (Repository) -> Interagir com o banco de dados.
